1. create directory for api 2. create documentation.html file 3. create server.js file that imports the http module and listens for requests on port 8080. 4. save all files & run 'node server.js' in terminal 5. head to browser and go to http://127.0.0.1:8080/ (you should see 'hello node') 6. publish repository to git https://docs.github.com/en/get-started/importing-your-projects-to-github/importing-source-code-to-github/adding-locally-hosted-code-to-github#adding-a-local-repository-to-github-using-git 7. delete server.js, create index.js, run 'npm init' + 'npm install --save express' + 'npm install --save body-parser' 7. git add . && git commit -m "commit message"
